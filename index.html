Bountiful Child Nutrition Tracker

There are a lot of child malnutrition nonprofits around the globe. It’s difficult to keep track of all the data for thousands of kids that are screened. This apps allows their data to be entered and saved, and tracked over time. It keeps records organized by both country name and community name.

Admin Users have the ability to create a new country, as well as access all communities serviced in a country. Clicking on a country shows a list of communities where screenings have taken place.

Within communities, Admin Users have the ability to review all children. Clicking on a community shows a list of children screened.

User can view an individual child. If a child has multiple screening instances, this app shows a graph of their weight over time.
